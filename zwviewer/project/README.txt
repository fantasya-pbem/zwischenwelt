compile and run in viewer/ dir with
make -C project && ./zwviewer

main is in project/source/shell/main.cpp

runtime path requirements :
needs (relative from viewer/) ../map.dat generated by mapout2.php
and everything inside ../gfx/

library requirements : 
apt-get install libglew-dev 
# or something like that is required for opengl extension headers
SDL-dev
opengl-dev

hardware requirements : 
uses the opengl multitexturing extension, must be supported by graphics-card,
but is quite common, even on older cards that do not even support shaders...
